복수 값 출력

    List 와 같은 복수 값 출력하는 방법
    - 1) 인덱스번호로 요소 하나하나 접근해서 출력
    - 2) th:each="item : ${List 또는 배열}"
      -> 향상된 for문 형태
      -> List 또는 배열 길이 만큼 반복
      -> 매 반복 시 마다 List 또는 배열의 요소를 차례대로 꺼내
         item 변수에 저장 (item 변수명은 자유롭게 작성)
  • 과일목록
  • 0번 인덱스 과일
  • 1번 인덱스 과일
  • 2번 인덱스 과일
