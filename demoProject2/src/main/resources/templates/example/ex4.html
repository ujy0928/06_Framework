타임리프 조건문 [1] - th:if / th:unless

    th:if = ${조건식}
    - 조건식이 ** true ** 인 경우에만
      해당 속성이 작성된 요소를 화면에 출력
  
    th:unless = ${조건식}
    - 조건식이 ** false ** 인 경우에만
      해당 속성이 작성된 요소를 화면에 출력

삼항 연산자

    - 타임리프 속성에 
      삼항 연산자( 조건식 ? 참인경우 : 거짓인경우 ) 작성 가능

삼항 연산자

Elvis 연산자 ( ?: )

    [작성법]

    값 ?: 값이 없을 때

    - 삼항 연산자에서 조건식 자리에 값만 작성
      (==, != 등의 연산자 사용 X)
  
    - 우변에는 값이 없을 때에 대한 값만 작성

    - 왼쪽 값이 존재하면 해당 값 출력, 없으면 우변을 출력

No-Operation 연산자 ( 값 ?: _ )

    - 왼쪽 값이 없을 경우 (== null)
      타임리프 코드를 해석하지 않는 연산자
    
    - 타임리프 코드 해석 X
      -> 일반 HTML 태그로 동작
      -> HTML 태그 사이 내용이 화면에 출력
